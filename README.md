---|---|---|---|---

快速排序 | O(n*log<sub>2</sub>n) |  O(n<sup>2</sup>) | O(log<sub>2</sub>n) | 不稳定
堆排序 | O(n*log<sub>2</sub>n)|O(n*log<sub>2</sub>n)|O(1)|不稳定
归并排序) | O(n*log<sub>2</sub>n) | O(n*log<sub>2</sub>n)|O(n)|稳定
计数排序 | O(n+m)|O(n+m)|O(n+m)|稳定
桶排序 | O(n)|O(n)|O(m)|稳定
